## PRY MI PRIMER PAGINA DINAMICA

### Estructura de carpetas

```
PRY_MI_PRIMER_PAGINA_DINAMICA/
├── index.html                # Página principal
├── public/
│   ├── css/                  # Hojas de estilo personalizadas
│   ├── fonts/                # Fuentes utilizadas por el sitio
│   ├── img/                  # Imágenes propias (ej. david.png)
│   ├── js/                   # Scripts personalizados
│   └── vendor/
│       ├── bootstrap-5/      # Archivos de Bootstrap (CSS/JS/map)
│       └── templade/         # Plantilla base con assets, css y js
└── views/
    ├── about.html            # Sección "Sobre mí"
    ├── contact.html          # Sección de contacto
    ├── portafolio.html       # Galería o portafolio
    └── welcome.html          # Página de bienvenida
```

### Cómo ejecutar el proyecto

1. Descarga o clona este repositorio.
2. Abre la carpeta `PRY_MI_PRIMER_PAGINA_DINAMICA`.
3. Haz doble clic en `index.html` o arrástralo a tu navegador favorito.

No se requieren servidores ni dependencias adicionales: es un sitio estático que funciona directamente en el navegador.
